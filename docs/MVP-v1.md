# MVP V1 — ProfilScan

## Objectif

Importer une base locale, prendre ou importer une photo, detecter les profils, compter, proposer la meilleure reference et afficher une image annotee.

## Inclus V1

- Import local dataprofils.js
- Stockage local IndexedDB
- Camera smartphone
- Import image
- Analyse automatique apres capture
- Detection multi-profils
- Meilleur resultat uniquement
- Score de confiance
- Image annotee
- Fonctionnement GitHub Pages

## Exclus V1

- Historique
- Correction manuelle
- Export CSV/PDF
- Login
- Base publiee dans GitHub
- IA / reseau de neurones
